## Chosen next three slices

2. IFD scoring filter (with explicit feasibility gate)
3. PDF source stage

## Core rationale
- Evol-Instruct is the biggest missing Tier A generation recipe
- IFD is the highest-leverage next quality signal after generation improves
- PDF is the highest practical-value source expansion

## Important note
The IFD slice is planned with a hard feasibility gate because the current client stack may not support true teacher-forced response scoring cleanly through the existing provider path. The plan explicitly says to stop and document the limitation rather than build a fake approximation.
