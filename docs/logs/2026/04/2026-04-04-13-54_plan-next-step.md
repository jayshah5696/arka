## Strong suggestion on sequencing inside Slice 1

Implement in this order:
1. **ConfigLoader**
2. **core data models** (`Record`, `StageContext`, `LLMOutput`)
3. **artifact/run directory layout**
4. **CheckpointManager (SQLite)**
5. **stage interface + PipelineRunner**
6. **OutputWriter**
7. **LLMClient** adapters

Reason: this gives you a runnable pipeline skeleton before you spend time on provider details.

These are mostly settled by the spec, but worth freezing in code before scaffolding:

- Package layout: `src/arka/...`
