## 2025-02-27 - API Key Leakage via StageLLMOverride / Vulnerability: Plaintext API key leaked to disk / Learning: Pydantic base models with exclude=True prevent this in config dumps, but nested configurations lacking it bypass the mitigation / Prevention: Always enforce `SecretStr` with `Field(exclude=True)` on *all* configurations dealing with credentials, not just the root/top-level ones.
