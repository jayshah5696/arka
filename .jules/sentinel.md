## 2025-02-28 - SecretStr Config Serialization / Vulnerability: Plaintext API keys leaked into serialized config files (`config.resolved.yaml`) / Learning: Custom `json_encoders` are deprecated in Pydantic V2 and do not prevent serialization when `model_dump(mode="json")` is called. `Field(exclude=True)` must be used to exclude secrets from dumps entirely. / Prevention: Use `Field(exclude=True)` for all sensitive configuration fields to ensure they never reach disk.
