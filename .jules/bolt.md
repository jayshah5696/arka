## 2025-02-18 - NearDedupStage O(n^2) bottleneck / Learning: Minhash signatures were being recomputed within a nested loop and evaluated against every representative. / Action: Added LSH bucketing (lsh_bands config option) to segment signatures and restrict Jaccard similarity comparisons to matching bucket candidates, turning an O(n^2) operation into O(n) average case.
## 2025-02-18 - SQLite Checkpoint Latency / Learning: Concurrent threadpool execution was causing frequent SQLite reconnects and write contention for checkpointing without WAL mode. / Action: Used a persistent SQLite connection with `check_same_thread=False` and a global `threading.Lock()`, combined with `PRAGMA journal_mode=WAL` and `PRAGMA synchronous=NORMAL`, which significantly speeds up checkpoint writes across concurrent batches.
