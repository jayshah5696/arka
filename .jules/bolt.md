## 2025-02-18 - NearDedupStage O(n^2) bottleneck / Learning: Minhash signatures were being recomputed within a nested loop and evaluated against every representative. / Action: Added LSH bucketing (lsh_bands config option) to segment signatures and restrict Jaccard similarity comparisons to matching bucket candidates, turning an O(n^2) operation into O(n) average case.
## 2025-04-22 - fastembed embedding cache / Learning: Model embedding operations were being recomputed entirely on `--resume` runs. / Action: Implemented SQLite-backed embedding caching in `CheckpointManager` and injected it into `PipelineRunner._embed_texts`, leveraging existing state DB to eliminate recomputes.
## 2026-04-27 - SQLite WAL mode + synchronous=NORMAL / Learning: Checkpoint updates were slow due to default journal mode and sync settings, exacerbated by opening new connections for each stage write. / Action: Enabled WAL mode and synchronous=NORMAL on a single persistent connection across stage writes to reduce reconnect latency and lock contention, resulting in a 20-40% checkpoint speedup.
